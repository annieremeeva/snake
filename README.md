## Обзор

Классическая игра Змейка в неоновом стиле.

### Цели

Во время разработки ставились следующие цели:

- Оптимальный вид в зависимости от устройства
- Игра в Змейку
- Оповещение об окончании игры
- Подсчет очков
- Ускорение после съедения яблока
